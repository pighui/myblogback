### 列表创建方式

```python
a1 = np.array([1, 2, 3]) #　创建一维数组
a1
```



```
array([1, 2, 3])
```



```python
a2 = np.array([
    [90, 80],
    [100, 90],
    [80, 100]
]) #　创建二维数组　-> 课程1，课程2的成绩
a2
```



```
array([[ 90,  80],
       [100,  90],
       [ 80, 100]])
```



```python
# 注意：数组的每个元素类型都保持一致
# 如果列表中的数据类型不同时，则会按 int->float->np.string
a3 = np.array(['disen', 20, 90.5])
a3
```



```
array(['disen', '20', '90.5'], dtype='<U5')
```



### 查看数组对象的属性

- dtype 数组对象的类型
- ndim 数组对象的维度
- size 数组的成员总数量
- shape 数组的形状
