if 后面跟的是条件表达式，条件表达式的结果为True或者False。

（1）如果if后面的条件是数字，只要这个数字不是0，python都会把它当做True处理

```python
if 0:
    return False
```
<!--more-->
（2）如果if后面跟的是字符串，则只要这个字符串不为空串，python就把它看作True

```python
if "":
    return False
```
